	Программа для фильмов (мультфильмов, короткометражек)

функции:
1. Хранения списка просмотренных фильмов
2. Оценки, комментарии, теги для фильмов
3. Актёры, режиссёр, жанры, композиторы, продюсеры, авторы сценария(идеи), продолжительность, оценка сайтов, ссылка

Данные по фильмам хранятся в базе.
Поиск по 2 и 3 пунктам.
CRUD для фильмов.



Отдельно для сериалов (мультсериалов): количество серий, количество сезонов.


Жанры: основные - боевик, вестерн, военный фильм, гангстерский фильм, детектив, детское кино, драма (криминальная, историческая, психологическая), исторический, киноальманах, комедия, комедия-драма, фантастика, мелодрама, музыкальный фильм, найденная плёнка, постапокалиптика, приключенческий фильм, псевдодокументальный фильм, производственный фильм, триллер, фильм категории Б (B-movie, Direct-to-video), фильм-биография, фильм-катастрофа, фильм-сказка, чёрный юмор,
